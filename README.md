# 👁️ Visual Aid — AI-Powered Mobile App for the Visually Impaired

An innovative mobile application that leverages Artificial Intelligence and Machine Learning to empower visually impaired individuals with real-time visual information. The app captures images/videos, uses an ML-powered chatbot to describe the visual content, and delivers the output as audio — enabling independence for blind users.

---

## 📌 Description

This project was developed as a software engineering initiative to bridge the accessibility gap for visually impaired individuals. The app harnesses the power of AI vision models and a machine learning chatbot to help blind users understand their surroundings by:

- Capturing images and videos via their mobile device
- Describing and interpreting the visual content using an ML-powered visual chatbot
- Converting chatbot responses into clear, comprehensible audio
- Allowing interactive Q&A about the captured visual content
- Integrating seamlessly with mobile assistive technologies

The goal is to provide a transformative tool that fosters independence and improves quality of life for visually impaired users.

## 🎯 Project Objectives

- Enable visually impaired users to capture images and videos using their mobile devices
- Implement a robust ML-powered visual chatbot that accurately describes captured content
- Transform chatbot responses into clear audio output for the user
- Facilitate an interactive chatbot interface for questions about visual content
- Ensure seamless integration with accessibility features on mobile platforms
- Enhance daily life by providing timely, relevant visual information
